## Install

**Install Dependencies**
```bash
pnpm add chroma-js
```

**Copy and paste the following code into your project.**

```tsx file=<rootDir>/src/registry/gradient-progress.tsx
```

## Props

| Prop | Type | Description |
|---|---|---|
| size | number | Width of a column, default = 4 |
| gap | number | Space between columns, default = 4 |
| startColor | string | any valid CSS color |
| endColor | string | any valid CSS color |
| percent | number | Progress bar percentage |
